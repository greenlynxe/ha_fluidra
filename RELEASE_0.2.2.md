# Fluidra Pool 0.2.2

Correctif de chargement pour certains appareils Fluidra.

## Changements

- Le endpoint `uiconfig` est maintenant optionnel.
- Une erreur HTTP 500 sur `uiconfig` ne bloque plus le chargement d'un appareil.
- Les entites continuent a etre creees depuis les composants Fluidra disponibles.
